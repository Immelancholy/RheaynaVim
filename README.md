# RheaynaVim

This is my NeoVim configuration for NixOS.
It uses the Nix Wrapper Modules neovim template:
[Template](https://github.com/BirdeeHub/nix-wrapper-modules/tree/main/templates/neovim)

You can run this configuration with:
```Bash
nix run github:Immelancholy/RheaynaVim#default
```
